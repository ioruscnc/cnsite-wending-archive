# cnsite-wending-archive

本仓库用于归档和发布多个独立的 HTML 页面。这些页面以静态形式保存，内容涵盖不同时期与主题的站点资料，不针对任何单一域名或具体网站。

## 项目简介

本项目旨在收集、整理并长期保存一批具有参考价值的 HTML 页面。所有页面均为独立文件，可直接在浏览器中打开查看。仓库本身不提供任何实时数据或动态服务，仅作为静态页面存档。

## 目录说明

```
/
├── pages/          # 存放所有归档的 HTML 页面
├── assets/         # 页面引用的静态资源（如图片、样式、脚本）
├── index.html      # 仓库首页（可选，提供页面索引）
└── README.md       # 本文件
```

- `pages/`：每个子目录或单个文件代表一个独立的归档页面，命名遵循日期或主题标识。
- `assets/`：资源文件按页面或类型组织，避免路径冲突。
- `index.html`：如果存在，则作为浏览归档内容的入口。

## 页面归档说明

- 所有页面均为静态 HTML，不依赖后端服务。
- 页面内容在归档时已尽可能保留原始结构、样式与文本。
- 部分页面可能因外部资源失效（如外部 CDN、字体、图片）而显示不完整，此为正常现象。
- 归档时间与原始页面发布时间可能不一致，内容不代表当前状态。

## 维护说明

- 本仓库不定期更新，新增页面或修正已有页面中的明显错误。
- 欢迎通过 Issue 提交页面建议或报告问题，但不保证所有请求均被处理。
- 不接受涉及商业推广、敏感内容或违反平台政策的页面提交。
- 仓库维护者保留对归档内容进行筛选、调整或删除的权利。

## 使用建议

- 可直接将仓库克隆到本地，通过浏览器打开 `index.html` 或 `pages/` 下的任意页面。
- 如需引用归档页面，建议注明仓库地址与归档日期。
- 请勿将本仓库内容用于违反法律法规或平台规则的目的。

本仓库内归档的页面内容遵循其原始来源的许可条款。如无特别声明，仓库整体结构及元数据采用 [MIT 许可证](LICENSE) 发布。
